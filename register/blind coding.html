BLIND CODING


Blind Coding tests your keyboard proficiency. It has three rounds.

Round 1:

Test your memory with keywords
Players will be shown slides with keywords from different languages for around 30 seconds. They need to repeat as many number of keywords as possible. Top 10-20 participants (depends on the number of participants) that are able to remember and type the most number of keywords in the given time will be qualified for the next round.



Round 2:

Re-type code
A code is provided and players need to re-type the given code in the provided time, without using the backspace key.



Round 3:

Trust your fingers
This round is similar to round 2 but re-typing of the code needs to be done in the provided time, without looking at the screen.
